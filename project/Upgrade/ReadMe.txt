V1.1.0 修改记录
1、不兼容V1.1.0版本；
2、支持存储云平台采集盒和I03T DEVICE ID；
3、支持TAS 4G模块和ZLG 4G模块；
4、支持U盘导入从云平台导出的DEVICE ID和SN码；
设备编号的格式:
I03M1-I03T1-1-000


从物联网平台导出的I03T及SN的device id的文件名称必须命名为i03m_dev_info.csv
从物联网平台导出的I03M的token的文件名称必须命名为
i03m_token.csv
然后把这两个文件拷贝到U盘的路径 U:\DEY\PARA\下面
5、同时支持I03T主从模式（1个电池组挂多个I03T形式，1个I03T为主，其他为从）、
   独立模式（每个电池组挂1个I03T）、和两种模式混搭；"
6、支持U盘导出SN码和DEVICE ID；
7、对外接口（上行1接口）支持波特率设置；
8、上行1口支持Modbus和私有协议
9、上行2口支持TAS 4G模块和ZLG 4G模块
10、优化SOC和SOH算法

此次更新增加了深度充放电次数，温度系数，电流系数，关联SOH计算值：
1：温度＜0时，温度系数temp_coeff=0.8；0~10°时，temp_coeff=0.9；温度10°~40°时，temp_coeff=1.0；＞40°时。temp_coeff=1.1；
2：电流系数cur_coeff = -0.05*cur_cap_coeff*10+1.05，其中cur_cap_coeff=实时电流/标称容量
3：充放电次数减少的SOH计算公式SOH=（（充电电流*时间+放电电流*时间）/（2*标称容量*temp*coeff）/充放电次数）+每月固定减少的SOH；
   充放电次数减少的SOH计算公式SOH=（充放电次数差值/充放电次数）+每月固定减少的SOH；
